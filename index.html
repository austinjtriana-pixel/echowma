How to Score Segments

  • Left-click a segment to cycle through scores
  • Right-click a segment to open a picker for direct score selection
  • With the picker open, press 0-5 to set a score, Esc to close
  • Use Set All buttons below the diagrams for bulk scoring

Navigation

  • After Case 5, you will see a review screen before final submission
